# User Management Web Application

## Technologies Used
- Python
- Flask
- HTML
- CSS
- SQLite

## Project Flow

1. User enters details in the form.
2. Flask receives the data.
3. Data is stored in SQLite database.
4. All records are fetched from the database.
5. Records are displayed in a table.

## Features

- Add User
- View Users
- Persistent Storage
- Simple User Interface
